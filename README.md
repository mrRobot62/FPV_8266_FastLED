# Building ring stands
All ring stands are build in the same manner. You need per stand
* Wodden bars 45x25mm
    * 1x 50cm
    * 1x 45cm
    * 2x 2cm
* Electro blank pipes M25 (in different length, like 2m), depending how hight your gate should be
